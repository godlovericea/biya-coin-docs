## 🚀 立即使用

### 本地测试

1. **方法1：直接打开**
   ```bash
   # 双击 index.html 或在浏览器中打开
   ```

2. **方法2：本地服务器**
   ```bash
   cd docs/md2html
   python -m http.server 8000
   # 访问 http://localhost:8000
   ```

### 在线部署

#### 🌟 最推荐：Vercel（5分钟完成）

```bash
# 1. 安装 Vercel CLI
npm i -g vercel

# 2. 在 md2html 目录下运行
cd docs/md2html
vercel login

# 3. 部署
vercel --prod
```

完成！你会得到一个网址，例如：
`https://md2html-viewer.vercel.app`

#### 其他平台

查看 `DEPLOY.md` 获取详细教程：
- Netlify（拖拽部署，超简单）
